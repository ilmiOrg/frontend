# 🚀 UniversityMatch - Educational Platform

## 📖 Overview

UniversityMatch is a **premium educational platform** that matches high-school students to universities, scholarships, and career paths in Central Asia and expanding markets. Built with modern React technology and featuring AI-powered matching, comprehensive internationalization, and premium user experience.

## ✨ Key Features

- **AI-Powered Matching**: Smart university and scholarship recommendations
- **Multi-Language Support**: English, Russian, Tajik, Uzbek, Kazakh, Kyrgyz, Urdu, Mandarin
- **Verification System**: Tiered badges for trust and authenticity
- **Social Features**: Alumni mentorship and student community
- **Application Management**: Timeline tracking with reminders
- **VR/AR Campus Tours**: Immersive virtual experiences
- **Cultural Fit Analysis**: Personalized compatibility scoring

## 🚀 How to Run

### Prerequisites
- Node.js 16+ 
- npm or yarn
- Modern browser with ES6+ support

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/yourusername/university-match.git
   cd university-match
   ```

2. **Install dependencies**
   ```bash
   npm install
   # or
   yarn install
   ```

3. **Start development server**
   ```bash
   npm start
   # or
   yarn start
   ```

4. **Open in browser**
   ```
   http://localhost:3000
   ```

### Build for Production

```bash
npm run build
# or
yarn build
```
